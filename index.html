Selected Projects

代表项目

针对大型语言模型的提示泄露攻击

面向 LLM 系统提示泄露风险，提出提示模板优化与后缀优化两类攻击方法，结合结构跳跃采样与候选包含对比机制生成高质量对抗输入。

论文已被 ISSRE 2026 接收。本人担任组长与核心成员，负责核心算法、攻击评估流程，以及 Methodology 和 Experiments 主要撰写。

面向大模型的细粒度代码脱敏方法 PiCo

国家级大学生创新训练项目。针对云端大语言模型处理代码时的隐私泄露问题，设计以用户提示为导向的代码语义脱敏框架，利用本地小语言模型构建标识符白名单。

参与框架设计与优化，负责代码标识符提取、关键词匹配模块、Stack Overflow 数据集处理、实验评估与技术文档。

中山大学珠海校区全景导览程序

基于 Go、Gin、MySQL 与 Redis 开发虚拟校园导览系统，支持 360 度全景展示、交互式地图导航与用户留言等功能。

作为后端负责人，主导系统架构设计和技术选型，实现用户认证、留言系统、场景切换、测试体系与 API 文档。

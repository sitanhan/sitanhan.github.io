# 韩思潭 GitHub 个人网站

这是一个可以直接部署到 GitHub Pages 的静态个人网站，内容聚焦项目经历。

## 修改内容

打开 `index.html`，可以继续补充或替换这些内容：

- GitHub 链接
- 邮箱地址
- 项目链接或论文链接
- 项目描述与个人贡献

## 部署到 GitHub Pages

1. 如果要作为个人主页，仓库应命名为 `anisnock.github.io`。
2. 把本文件夹里的所有文件上传到仓库根目录。
3. 进入仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main` 和 `/root`，保存。
6. 等待几分钟后，访问 `https://anisnock.github.io`。
